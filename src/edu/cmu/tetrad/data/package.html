Implements a collection of coltData containers, including containers for typed coltData as well as containers for cluster and knowledge information. Data containers are stored as lists of columns of varying underlying types, where each column is supplied with a variable indicating how the coltData in that column is to be interpreted. Knowledge and cluster information is stored in appropriately wrapped maps. Additional classes are included to perform certain types of manipulations on coltData sets or to store collections of coltData sets.
